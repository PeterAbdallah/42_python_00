# Python Functions & Basics
## Overview

This project covers the essentials of Python functions and basic operations: writing functions, conditionals, printing, input, string methods, recursion, and nested functions.

### Functions

- Defined using def.
- Can receive parameters (inputs) and return values.
- Reusable and help organize code.

### Indentation

- Python uses spaces to define blocks.
- Indent consistently for functions, loops, and conditions.

### Conditional Statements

- if → run code if condition is true.
- elif → check another condition if previous if failed.
- else → run code if all previous conditions are false.

### Printing and Input

- print() → display output.
- input() → read input (always a string).
- Use int() to convert input to numbers if needed.

### String Methods

- Strings have built-in methods like .capitalize() to format text.
- Can be used directly in print() without changing the original string.

### Recursion

- A function can call itself to solve smaller versions of a problem.
- Always include a base case to stop recursion.
- Work can happen before or after the recursive call.

### Nested Functions

- Functions can be defined inside other functions.
- Useful for helper functions that only exist inside the main function
